# CCNA Materials

🌍 **English** | [Magyar](README.md)

A collection of study materials for the **Cisco CCNA** certification and Cisco Networking Academy (NetAcad) courses — lecture slides, module presentations, and practical guides covering routing, switching, and wireless networking.

## 📂 Repository structure

| Folder | Contents |
|--------|----------|
| **`2008-as PPT-k`** | Topic-based slides from the older (2008-era) CCNA curriculum: ACLs, DHCP, NAT, VLANs, inter-VLAN routing, static and dynamic routing, single-area OSPF, and switched networks. Includes a Hungarian-language NAT presentation. |
| **`ENSA-2016 PPT`** | Module presentations for the **ENSA** course (Enterprise Networking, Security, and Automation, Modules 1–14), plus a PDF on point-to-point connections. |
| **`2019-es PPT-k SRWE v7`** | Topic slides for the CCNAv7 **SRWE** course (Switching, Routing, and Wireless Essentials, 2019): basic device configuration, VLANs, inter-VLAN routing, STP, EtherChannel, DHCPv4, SLAAC & DHCPv6, FHRP, LAN and switch security, WLAN concepts and configuration, routing concepts, and static route troubleshooting. |
| **`SEWE v7.0 EN`** | English-language **SRWE v7.0** module presentations (Modules 1–16). |
| **`cisco_password_recovery`** | Step-by-step Word guides for password recovery on Cisco **routers** and **switches**. |

## 🎓 About the CCNA v7 curriculum

The current CCNA (v7) path consists of three courses:

1. **ITN** – Introduction to Networks
2. **SRWE** – Switching, Routing, and Wireless Essentials *(covered in this repo)*
3. **ENSA** – Enterprise Networking, Security, and Automation *(covered in this repo)*

## 🚀 How to use

- Browse the folders and download the `.pptx` files you need — GitHub can preview most presentations directly in the browser.
- Open the slides with Microsoft PowerPoint, LibreOffice Impress, or Google Slides.
- Practice the configurations in **Cisco Packet Tracer** or on real lab equipment.

## ⚠️ Disclaimer

These materials are shared **for educational purposes only**. Cisco, CCNA, and Cisco Networking Academy course content are the property of Cisco Systems, Inc.
Note: This repository does not contain the full courses — only materials that are publicly available on the internet (though you have to search pretty hard to find them), collected and stored here in one place.



# CCNA tananyagok

🌍 [English](README.md) | **Magyar**

Tanulási segédanyagok a **Cisco CCNA** vizsgához és a Cisco Hálózati Akadémia (NetAcad) kurzusaihoz — előadásdiák, modulprezentációk és gyakorlati útmutatók a forgalomirányítás (routing), a kapcsolás (switching) és a vezeték nélküli hálózatok témaköreiben.

## 📂 A repó felépítése

| Mappa | Tartalom |
|-------|----------|
| **`2008-as PPT-k`** | Témakörönkénti diák a régebbi (2008-as) CCNA tananyagból: ACL-ek, DHCP, NAT, VLAN-ok, VLAN-ok közötti forgalomirányítás, statikus és dinamikus routing, egyterületes OSPF, kapcsolt hálózatok. Magyar nyelvű NAT-prezentációt is tartalmaz. |
| **`ENSA-2016 PPT`** | Az **ENSA** kurzus (Enterprise Networking, Security, and Automation) modulprezentációi (1–14. modul), valamint egy PDF a pont-pont összeköttetésekről. |
| **`2019-es PPT-k SRWE v7`** | Témakörönkénti diák a CCNAv7 **SRWE** kurzushoz (Switching, Routing, and Wireless Essentials, 2019): alapszintű eszközkonfiguráció, VLAN-ok, VLAN-ok közötti forgalomirányítás, STP, EtherChannel, DHCPv4, SLAAC és DHCPv6, FHRP, LAN- és kapcsolóbiztonság, WLAN-alapok és -konfiguráció, forgalomirányítási alapok, statikus útvonalak hibaelhárítása. |
| **`SEWE v7.0 EN`** | Az **SRWE v7.0** kurzus angol nyelvű modulprezentációi (1–16. modul). |
| **`cisco_password_recovery`** | Lépésről lépésre útmutatók (Word) Cisco **routerek** és **switchek** jelszó-visszaállításához. |

## 🎓 A CCNA v7 tananyagról

A jelenlegi CCNA (v7) képzés három kurzusból áll:

1. **ITN** – Introduction to Networks (Bevezetés a hálózatokba)
2. **SRWE** – Switching, Routing, and Wireless Essentials *(ebben a repóban megtalálható)*
3. **ENSA** – Enterprise Networking, Security, and Automation *(ebben a repóban megtalálható)*

## 🚀 Használat

- Böngészd a mappákat, és töltsd le a szükséges `.pptx` fájlokat — a GitHub a legtöbb prezentációt közvetlenül a böngészőben is meg tudja jeleníteni.
- A diák megnyithatók Microsoft PowerPointban, LibreOffice Impressben vagy a Google Diákban.
- A konfigurációkat érdemes **Cisco Packet Tracerben** vagy valós laboreszközökön gyakorolni.
